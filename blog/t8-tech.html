Web Security

Web Security is to protect website from attacking, changing content in web. People who didn't work with the web security will not pay attention on the security issue a web has or not. However, for most time, web security issue exists and web security is important and necessary to protect the web.

Basic Web Security Issues

1. For most web design, developer only consider the basic customers without thinking more about webpage holes.
2. The web security is too behind so that it didn't work.
3. Hacker attacked web and changed web but web cannot find it.
4. Although we realize the issue, we cannot solve it.
