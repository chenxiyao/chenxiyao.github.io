Arrays and Hashes

This time, let's think learn something about Ruby. When we want to run some single thing, we can code directly like age=25 in Ruby. However, if we want to run the whole class's students' ages. Or we want to code a person's all information like first name, last name, age. At this time, we need to use arrays and hashes. However, arrays and hashes are different, they have different use and be used in different place

Arrays

Using arrays, we can express many same things in one time. For examples, there are five people in one group, I want to show their ages. So it can be
age1=10
age2=11
age3=12
age4=13
age5=14
But we can express them easier like
age=[10,11,12,13,14]
then at this time, age1=age[0]

There are many methods using in arrays. age.push(15) means add the element 15 at the end of array age. And there are many other methods that can help to solve some problems much more easily.

Hashes

Hash also can express different things in one time, but it's different from arrays. For example, we also want ot show the five people's age.
Then using hash:
age = {
age1 => 10,
age2 => 11,
age3 => 12,
age4 => 13,
age5 => 14
}

We can also code hash like this age[age1]=[10]

As the beginner, we first need to know when to use arrays and hases and the basic use of them. Then the important part that sometimes they are similar but we need to know they are different.
