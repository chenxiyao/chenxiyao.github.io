Class Methods

Now, let's learn something new in Ruby. We know we can use method to define something. However, when we have many method to use together with a total thing. We can use class to implement it.

A simple example

There's a simple example about die. We want to code how many sides for the die and what's the value for a roll:
die.sides will return 6
die.roll will return a random number from 1 to 6

Some important

Now we know some basic knowledge about class. Here we need to know some important definitions.
The first is variables. There are four different kinds of variables: local variables, instance variables, class variables, and global variables.

Local Variables: Local variables are defined in a method. That means local variables are not available out of the method. Local variables begin with a lowercase letter or _.

Instance Variables: Instance variables are available across all methods in a class. Instance variables begin with (@).

Class Variables: Class variables make a variable accessible to ALL instances of the class. Class variables begin with @@.

Global Variables:Global variable is available across classes. The global variables start with ($).

Another important part is the initialize method. Like in the example, at the beginning of a class, we always write the initialize method to initialize the variables.

Class is a hard part to learn. I believe we need to more practice to be familiar with class first. And then we can learn some more use with class.
