SQL Basics

  • SELECT
  • WHERE
  • ORDER BY

SELECT

  1. Select all data for a table

    2. For example, we want to select all data for table "states".
    SELECT * FROM states;

  2. Select the columes for a table.

    3. For example, we want to select the "state_name" and "population" for table "states".
    SELECT state_name, population FROM states;

WHERE

  1. Select the exact data from a table

    2. For example, we want to select the 'state_name's in table "states" where "region" is 7.
    SELECT state_name FROM states
    WHERE region_id=7;

ORDER BY

  1. Order by Ascending

    2. For example, we want to select "state_name" and "population" for table "states" by the order of "population" ascending.
    SELECT state_name, population FROM states
    ORDER BY population;

  2. Order by Descending

    3. For example, we want to select "state_name" and "population" for table "states" by the order of "population" descending.
    SELECT state_name, population FROM states
    ORDER BY population DESC;
